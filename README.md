# Surfboard
WAV file reading and writing in Rust

Currently tested with Rust 1.5.0 on OSX El Capitan

```
Usage: surfboard -r <source>
       surfboard -w <source> <dest>
       surfboard -h

Options:
    -r, --read   Parse file.
    -w, --write  Write data to file.
    -h, --help   Show this message.
```
